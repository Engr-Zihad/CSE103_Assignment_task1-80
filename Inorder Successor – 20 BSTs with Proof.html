📖 Definition & Rules

In a Binary Search Tree (BST), the Inorder Successor of a given node is the next node that would be visited in an inorder traversal (Left → Root → Right).

Rule 1 If the node has a right subtree, the successor is the leftmost node in that right subtree. Rule 2 If the node has no right subtree, the successor is the lowest ancestor for which the node is in its left subtree (climb up until you go left).
